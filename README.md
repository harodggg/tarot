# 🔮 塔罗牌占卜 Tarot Divination

基于韦特塔罗（Rider-Waite-Smith）体系的在线塔罗牌占卜工具。

## 功能
- 78 张完整塔罗：22 张大阿卡纳 + 56 张小阿卡纳
- 正逆位随机判定
- 8 种牌阵：单张牌 / 三牌阵 / 四季牌阵 / 五牌阵 / 关系牌阵 / 七牌阵 / 六芒星牌阵 / 凯尔特十字
- 韦特系 PNG 牌面图片
- 自动翻牌动画 + 粒子特效
- 桌面端和移动端适配
